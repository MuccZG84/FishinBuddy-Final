# FishingBuddy 🎣

FishingBuddy je jednostavna web aplikacija za praćenje ribolovnih lokacija, ulova i vremenskih uvjeta. Izgrađena je pomoću React-a, HTML-a i JavaScript-a.

## 🚀 Tehnologije

- React
- JavaScript
- HTML5
- CSS3
